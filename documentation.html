PhaseOne Viewer Documentation

Overview

The PhaseOne Viewer provides a interface to explore and interact with datasets dynamically. This documentation outlines the new features, changes, and usage instructions for the viewer.

New Features

  • Dynamic Info Popup: Displays detailed information about datasets, including video embeds and actionable buttons.
  • Buttons for Download and Case Stories: Links for dataset downloads and case studies appear dynamically if available.
  • YouTube Video Embed: Displays a video related to the dataset, if configured in the datasets.js file.
  • Improved URL Parameters: Supports toggling the info popup using &info=true in the URL.
  • Interactive Close: Info popup and menu close automatically when clicking outside their respective areas.

How to Use

Dataset Filtering

Use the following URL parameters to filter datasets:

  • ?filter=all - Shows all datasets (default).
  • ?filter=p3 - Shows only P3 datasets.
  • ?filter=p5 - Shows only P5 datasets.

Info Popup

The info popup can be toggled dynamically:

  • Add &info=true to the URL to display the info popup on page load.
  • Click the Info button in the top bar to open it manually.

Embedding the Viewer

The viewer can be embedded on any webpage using an iframe. Below are the steps and examples:

Basic Embedding

Use the following iframe code to embed the viewer:

<iframe src="https://showmethedata.phaseone.com?filter=p3" 
    width="100%" height="500" frameborder="0"></iframe>

Advanced Embedding with Parameters

Customize the viewer experience by adding URL parameters:

  • ?filter=p3: Display only P3 datasets.
  • ?filter=p5: Display only P5 datasets.
  • ?info=true: Automatically open the info popup.
  • ?id=example-dataset: Directly display a specific dataset.

Example with multiple parameters:

<iframe src="https://showmethedata.phaseone.com?filter=p5&id=Town-Roof" 
    width="100%" height="600" frameborder="0"></iframe>

Styling and Responsiveness

To make the iframe responsive, use the following CSS:

.responsive-iframe {
    position: relative;
    width: 100%;
    padding-bottom: 56.25%;
    height: 0;
}

.responsive-iframe iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}
